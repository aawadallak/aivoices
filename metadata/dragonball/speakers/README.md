# Dragonball Speakers

Use this area for speaker-related artifacts.

- `references/<speaker>/`: reusable voice references for future runs
- `references/<speaker>/approved/`: validated reference clips
- `references/<speaker>/pending/`: clips that still need review
- `runs/<run-id>/<speaker>/extracted-clips/`: already-extracted clips for a speaker in a specific run
- `runs/`: outputs from diarization or speaker-specific extraction runs
